TransactionApi
Transaction's api made in java using springboot

Stack:

Java 11;
Spring Boot;
Docker;
Swager;

Urls:

http://localhost:8080/ -Transaction's api;
http://localhost:8080/swagger-ui.html - Swagger da aplicação;

Inicializando a Api:

Executar o comando docker-compose up na pasta da aplicação;
